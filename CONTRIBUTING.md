# Contributing

leaning-rctへの改善提案を歓迎します。Issueでは、対象章、期待する説明または挙動、現在の説明または挙動を明記してください。

## Pull request

1. Node.js 24以降とpnpm 10を用意します。
2. `pnpm install`を実行します。
3. 実装変更には単体テスト、教材変更には該当章の確認を追加します。
4. `pnpm check`が成功することを確認します。

教材の章は「今回の到達点」「前提知識」「小さな実装」「Playground確認」「テスト」「本家Reactとの対応」の順を基本とします。未完成機能は公開APIへ追加せず、WIPであることを明示してください。

他の教材・記事・ソースコードを参照した場合は、本文の近くに出典を記載します。文章や図版の転載ではなく、自分の言葉と独自の例で説明してください。
